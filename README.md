pyTerrainTree is a Python wrapper around the [`Terrain_Trees` C++ library]
(https://github.com/FellegaraR/Terrain_Trees).

```shell
git clone https://github.com/zacharyburnett/pyTerrainTree
cd pyTerrainTree
pip install .
pip wheel . -w dist --no-deps
```
